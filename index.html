My Projects

raceVision

For part 1 of my final year group software engineering project, we built raceVision. The application could display the races for the 2018 America's Cup as they happened live. The code was written mostly in Java, with some use of JavaScript to communicate with Google's Maps API

High Seas

For part 2 of my final year group software engineering project, we built High Seas. Due to the 2018 America's Cup finishing half way through the year, we were instructed to pivot the product to a multiplayer sailing game whilst retaining much of the existing Java code base. Many challenges presented themselves with this approach, and this provided an interesting learning experience.

Ride Sharing application

An individual software engineering assignment I worked on was to build a basic ride sharing application in Java. The application allows a user to register their cars, set up trips to or from University with custom stop points, share these trips and also book rides on other shared trips.
